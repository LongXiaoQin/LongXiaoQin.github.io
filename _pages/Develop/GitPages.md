### 1.选择主题

```http
http://jekyllthemes.org/
```



### 2.Fork

- 点击Homepage进入github代码仓库，点击Fork

- Respository name 填写为[username].github.io，点击create fork

- 点击settings->Pages，Branch选择master，点击Save

### 3.选择构建方式

- Build and deployment 选择GitHub Actions

- 选择Jekyll，点击Configure

- 点击 commit changes
